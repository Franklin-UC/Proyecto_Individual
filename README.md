# Proyecto Web Personal - Franklin Ulloa
Este proyecto integra frontend, backend y base de datos.

## Requisitos
- Python 3.x
- MySQL
- Librería: `pip install mysql-connector-python`

## Instrucciones de ejecución
1. Ejecutar el script SQL en MySQL para crear la tabla `mensajes`.
2. Instalar el conector: `pip install mysql-connector-python`.
3. Iniciar el servidor: `python server.py`.
4. Abrir en el navegador: `http://localhost:8000/index.html`.

## Funcionalidades Implementadas
1. **Frontend**: 5 páginas con CSS personalizado.
2. **JS**: Validación de formulario y efectos dinámicos en menú.
3. **Backend**: Manejo manual de rutas y POST en Python puro.
4. **BD**: Almacenamiento y consulta de mensajes en MySQL.
